1. Grundlagen des Rechnungswesens

Rechnungswesen: Ein System zur Erfassung, Überwachung und Auswertung aller Geld- und Leistungsströme im Unternehmen.

Wichtige Begriffe: Fibu vs. KLR

Deutsch (Begriff) Russisch (Значение)

2. Finanzbuchhaltung (Bilanz)

Bilanz: Eine Gegenüberstellung von Vermögen (Aktiva) und Kapital (Passiva) zu einem Stichtag.
Formel: Summe Aktiva = Summe Passiva

Struktur der Bilanz

Aktiva (Mittelverwendung) Passiva (Mittelherkunft)
Anlagevermögen (AV)
Langfristig: Grundstücke, Maschinen, Fuhrpark. 		Eigenkapital (EK)
Geld der Eigentümer.
Umlaufvermögen (UV)
Kurzfristig: Vorräte, Forderungen (Kunden), Bank, Kasse. 		Fremdkapital (FK)
Schulden: Darlehen, Verbindlichkeiten (Lieferanten).
Aufgabe: Bilanz aufstellen & Kennzahlen

Gegeben: Fuhrpark 410k, Gebäude 7.200k, Ausstattung 50k, Vorräte 30k, Forderungen 110k, Wertpapiere 550k, Kasse 12k, Bank 180k.
Passiva: Darlehen 7.800k, Verbindlichkeiten 60k.

1. Eigenkapital berechnen:

Summe Aktiva (Vermögen) = 410+7.200+50+30+110+550+12+180 = 8.542.000 €
Summe Fremdkapital (Schulden) = 7.800 (Darlehen) + 60 (Verb.) = 7.860.000 €
Eigenkapital = Aktiva - Fremdkapital = 8.542.000 - 7.860.000 = 682.000 €

2. Kennzahlen berechnen:

Eigenkapitalquote: (EK / Gesamtkapital) * 100
(682.000 / 8.542.000) * 100 = 7,98 % Показывает финансовую независимость (здесь низкая).
Liquidität 1. Grades: (Flüssige Mittel / Kurzfr. Verb.) * 100
Flüssige Mittel = Kasse (12k) + Bank (180k) = 192.000 €
Kurzfr. Verb. = 60.000 €
(192.000 / 60.000) * 100 = 320 % Мы можем оплатить текущие долги 3 раза (очень хорошо).

3. Kostenrechnung (KLR)

Break-even-Point (Gewinnschwelle): Erlöse = Kosten. Gewinn = 0.
Deckungsbeitrag (DB): Verkaufspreis - variable Kosten.
Aufgabe: Break-even-Analyse (Nachttischlampen)

Daten: Preis 12,50 €, variable Kosten 7,00 €, Fixkosten 330.000 €.

1. DB pro Stück = 12,50 - 7,00 = 5,50 €
2. Break-even-Menge = Fixkosten / DB = 330.000 / 5,50 = 60.000 Stück

Ab 60.000 Stück macht die Firma Gewinn.

Aufgabe: Effektiver Stundensatz (Personal)

Frage: Interne Kraft (140.000 €/Jahr) vs. Externe Kraft (85 €/h).

Daten Intern: 260 Tage/Jahr, 30 Urlaub, 10 Krankheit/Feiertage. 7,8 h/Tag.

1. Produktive Tage = 260 - 30 - 10 = 220 Tage
2. Produktive Stunden = 220 * 7,8 = 1.716 h
3. Stundensatz = 140.000 € / 1.716 h = 81,59 €/h

Entscheidung: Intern (81,59 €) ist günstiger als Extern (85,00 €).

Aufgabe: Mischkalkulation (Hotline Minutensatz)

Ziel: Einen einheitlichen Minutenpreis für Kunden finden, der alle Kosten deckt.

Kosten pro Schicht:
- Bereitschaft (Fix): 120€ (1st Lvl) + 150€ (2nd) + 200€ (3rd) = 470 €
- Gespräche (Var): 75€ + 100€ + 75€ = 250 €
- Gesamtkosten = 720 €

Gesamtminuten: 150 + 100 + 50 = 300 Minuten.

Preis = Gesamtkosten / Gesamtminuten = 720 / 300 = 2,40 €/Min

6. Leasing

Leasing: Miete statt Kauf. Liquidität wird geschont, aber Gesamtkosten meist höher.
Aufgabe: Leasing oder Kauf?

Optionen: Kauf 300.000 € sofort ODER Leasing 6.000 €/Monat (5 Jahre).

1. Leasing gesamt: 6.000 * 60 Monate = 360.000 €
2. Kauf gesamt: 300.000 €
Diff: 60.000 €.

Entscheidung: Kauf ist rechnerisch günstiger. Leasing wird gewählt, wenn man Liquidität schonen will.

7. Entscheidungen & Investitionen

Make-or-Buy: Selbst herstellen oder einkaufen? Break-even-Analyse hilft.
Nutzwertanalyse: Punktesystem für Entscheidungen (nicht nur Preis, auch Qualität).
Aufgabe: Make-or-Buy (Lizenzen)

Daten: Fremd 25€/Jahr (250€ in 10 Jahren). Eigen 1.005.000€ (Fix).

Gleichung: Kosten Fremd = Kosten Eigen
250 * Menge = 1.005.000
Menge = 1.005.000 / 250 = 4.020 Stück

Ab 4.021 Stück ist Eigenentwicklung günstiger.

Aufgabe: Nutzwertanalyse (CAD Systeme)

Prinzip: Gewichtung * Punkte = Ergebnis.

Beispiel Unternehmen 2:
Grafikkarte (Gewicht 20, Note 2) -> 40 Pkt
RAM (Gewicht 25, Note 3) -> 75 Pkt
Monitor (Gewicht 40, Note 4) -> 160 Pkt
Preis (Gewicht 15, Note 4) -> 60 Pkt
Summe = 355 Punkte (Sieger)

8. Komplexe Aufgaben (Mischung)

Aufgabe: Kostenkalkulation Schließsystem

Gegeben: 65 Zylinder (512€), 80 Transponder (39€), Gerät (2300€). Lizenz 349€/Jahr für 5 Jahre.

1. Hardware (Einmalig): (65*512) + (80*39) + 2300 = 33.280 + 3.120 + 2.300 = 38.700 €
2. Lizenz (5 Jahre): 349 * 5 = 1.745 €
3. Netto gesamt: 38.700 + 1.745 = 40.445 €
4. Brutto (x 1,19): 48.131,55 €
Aufgabe: Monatskosten CAD-Plätze

Gegeben (5 Plätze): Monitor 450€ (4 Jahre), PC 720€ (3 Jahre). 5% Rabatt. Software 50€/Monat/Platz. Wartung 1200€/Jahr.

1. Anschaffung Netto (5 Stk): Monitor 2.137,50€, PC 3.420€.
2. Abschreibung/Monat:
- Monitor (48 Monate): 2.137,50 / 48 = 44,53€
- PC (36 Monate): 3.420 / 36 = 95,00€
3. Laufende Kosten: Software (250€) + Wartung (100€).
Summe: 44,53 + 95 + 250 + 100 = 489,53 € / Monat
